# 01 Base 11.par

Quick base model check from the BET 2026 input repository using `11.par`.

Use `stepwise-config.R` in the repo root to change labels, input files,
selected folders, or evaluation counts. To make this folder fully manual, place
the MFCL files in `steps/01-base-11par/model/` using the usual names
(`bet.frq`, `11.par`, `bet.tag`, `mfcl.cfg`, and related files).
